Etude sur les proximits entre rdactions

On peut visualiser des proximits entre rdactions du point de vue des journalistes qui appartiennent plusieurs rdactions (ou qui ont volu dune rdaction une autre). On cre donc un tableau de contingence croisant les rdactions entre elles, et qui donne dans une case le nombre de journalistes qui appartiennent simultanment la rdaction i et la rdaction j.

load("../data/journalist-and-description.RData")
don <- probable.journalist[,-(1:6)]
don[don>0] <- 1  ## recode les valeurs du tableau suprieure ou gale  2
don <- don[which(apply(don,1,sum)>1),] ## supprime journaliste dans une seule rdaction
don <- t(don)
don <- don[sort(rownames(don)),] ### tri des rdactions par ordre alphabtique
media <- c("J","TV","TV","TV","J","J","Radio","J","Radio","J","J","J","J","J","J","J","J","J","J","J","J","J","J","J")
national <- rep("y",nrow(don))
national[c(5,6,10,13,19,20,22,24)] <- "n"
select <- which(apply(don,1,sum)>3) ## supprime les rdactions qui ont moins de 3 journalistes
media <- media[select]
don <- don[select,] 
distRedaction=don%*%t(don)
resCA=CA(distRedaction,graph=FALSE)
plot(resCA,invisible="col",cex=.7)
